Issue
I am trying to upload a new presentation providing an old URL_Path. Getting an error "Duplicate URL exists" though I have already deleted the old presentation.
Environment
  • Adobe Connect Licensed 9.X

Cause
  • Old URL is still active in the database.

Resolution
  • Disable the URL from PPS_SCOS table from the Connect Database using the below query:
update pps_scos
set disabled = GETDATE()
where sco_id = XXXX

(Note: XXX is the sco ID of the old presentation.)
